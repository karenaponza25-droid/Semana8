# Semana8
Actividad de la Semana 8: Linux, terminal, código abierto y seguridad digital

Este repositorio contiene el desarrollo de la actividad correspondiente a la Semana 8 del curso. En esta práctica se abordan conceptos fundamentales sobre sistemas operativos de código abierto, uso de la terminal Linux y buenas prácticas de seguridad digital para la protección de plataformas tecnológicas comunitarias.

Se incluyen las respuestas a las preguntas teóricas, ejercicios prácticos en Linux, recomendaciones de seguridad y una reflexión sobre la importancia de proteger la información en entornos digitales.
